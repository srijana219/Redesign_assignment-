# Flutter Redesigned Screen Assignment

 Overview
This project is a redesigned version of an existing Flutter screen based on the provided Figma design.  
It preserves the existing *Boost component, implements a modern responsive UI, and follows clean widget composition principles.

 Features
-  Redesigned layout from Figma  
-  Responsive for all screen sizes (mobile, tablet, web)  
-  Separated UI into reusable widgets: Header, Boost, ContentCard, and ActionRow  
-  Organized file structure under /lib/screens/  
-  Clean and maintainable code  
-  Added image assets for a modern visual look
